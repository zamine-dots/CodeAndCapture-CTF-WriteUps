# Code&Capture CTF — Challenge Writeups

Thanks for checking out this repository.

I hope you enjoyed the challenges and had fun solving them.

This repo contains the **official writeups and source material only for the challenges that were successfully solved during the event**.

Some challenges are intentionally not included here because they were left unsolved, and I’d like to keep them available for future events, practice sessions, or remastered releases.

## Notes

These challenges were created for **Code&Capture**, with the goal of mixing:

* realistic vulnerabilities
* classic CTF techniques
* beginner-friendly learning paths
* enough depth for experienced players

If you solved them: well played.

If you're reading the writeups to learn, I encourage you to rebuild the challenges locally and reproduce the exploitation steps yourself.

Thanks for participating and supporting the event.
